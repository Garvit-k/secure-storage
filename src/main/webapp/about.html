About Company


Data stored over cloud and flow through the network in the plain text format is a security threat. This application proposes a method that allows users to store and access the data securely from cloud storage. It also guarantees that no one except the authenticated user can access the data nor the cloud storage provider. This method ensures the security and privacy of data stored on the cloud. A further advantage of this method is that if there is a security breach at the cloud provider, the userâ€™s data will continue to be secure since all data is encrypted. Users also need not to worry about cloud providers gaining access to their data illegally. Cloud computing gives a lot of benefits with low cost and of data accessibility through the Internet. Ensuring the security of the cloud the main goal is to securely store and access data in a cloud that is only controlled by the owner of the data. This application is based on cloud computing using hybrid cryptography algorithms such as AES .
